🪼 Misi 3: Konser Interaktif! 🪼

🎯 Tujuan Misi:

  • Membuat sprite yang dapat merespons input keyboard
  • Menggunakan blok Events untuk mendeteksi interaksi pengguna
  • Membuat sprite yang dapat merespons suara keras dari mikrofon

🎤 Blok Ajaib Hari Ini:

🔹 Dari Peristiwa (Events):

when [space] key pressed

→ Memulai aksi saat tombol tertentu di keyboard ditekan.

when backdrop switches to [backdrop]

→ Sprite akan bereaksi saat latar panggung berubah.

when loudness > (10)

→ Sprite bisa merespons suara keras dari mikrofon/alat perekam (seperti tepuk tangan).

🔹 Ditambah blok sebelumnya untuk mendukung cerita:

say "..." for (2) seconds
switch costume to [costume]
switch backdrop to [backdrop]
Dari Looks

start sound (Singer2)
Dari Sound atau suara lain

🎬 Tantangan Sutradara:

Aturlah script agar sprite-mu bisa tampil layaknya bintang konser:

  1. Saat Tombol Spasi Ditekan:
    • Sprite akan berkata "Yuk, bernyanyi!"
    • Kostumnya berganti menjadi jellyfish-d
    • Latar panggung berubah menjadi underwater 2
  2. Saat Latar Berubah:
    • Ketika latar panggung telah berganti ke underwater 2, sprite akan memainkan suara singer2
  3. Saat Ada Suara Keras dari Pengguna:
    • Sprite akan terus-menerus "mendengarkan" suara dari mikrofon
    • Jika mendeteksi suara keras (misalnya, loudness > 60), sprite akan memainkan suara cheer

🎯 Target:

Sprite-mu bisa bereaksi berbeda sesuai interaksi atau input yang kamu berikan (keyboard atau suara), membuat konser terasa hidup dan interaktif.

🎁 Tantangan Bonus:

Kamu bisa menambahkan beberapa blok when [key] pressed untuk tombol-tombol huruf yang berbeda (misalnya A, S, D, F). Atur agar setiap tombol memainkan suara alat musik yang unik, seolah-olah Anda sedang memainkan piano atau drum.

🌟 Tips:

Buat sprite-mu tepat di tengah-tengah sebelum bernyanyi
