🧠 Misi 5: Kuis Perkalian Satu Digit

🎯 Tujuan Misi

Menggunakan blok Control, Sensing, Variables, dan Operators untuk membuat kuis perkalian sederhana dengan soal acak dan respon jawaban.

🧩 Blok yang Digunakan

Control

repeat [ ] if [ ] then … else

Sensing

ask [ ] and wait answer

Variables

set [variabel] to [ ]

Operators

pick random [ ] to [ ] [ ] * [ ] [ ] = [ ] join [ ] [ ]
⚔️ Tantangan

Buat program Scratch yang:

  1. Memberikan 3 soal perkalian acak (1–9).
  2. Memberikan respon berbeda untuk benar dan salah.
🎯 Target
  • Program berjalan otomatis ketika bendera hijau diklik.
  • Semua soal perkalian muncul secara acak.
  • Respon benar dan salah muncul sesuai kondisi.
⭐ Tantangan Bonus
  • Ubah jumlah soal dari 3 menjadi lebih banyak.
  • Modifikasi teks pembuka dan respon sesuai kreativitas.
💡 Tips (Alur Program)
Start (klik bendera hijau)
Tampilkan instruksi awal (say pembuka)
Ulangi 3 soal
Ambil Angka1 acak (1–9)
Ambil Angka2 acak (1–9)
Tanyakan: "Berapakah hasil dari Angka1 x Angka2?"
Cek jawaban
Jika benar → suara Tada + tampilkan "Benar!"
Jika salah → suara Trombone + tampilkan "Salah! Jawaban yang benar adalah [hasil]"
Selesai
